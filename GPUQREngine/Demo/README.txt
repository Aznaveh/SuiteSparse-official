This GPUQREngine/Demo directory contains a simple demo of the GPUQREngine
package.  "make demo" in the GPUQREngine folder will compile and run the demo
on a set of randomly constructed dense matrices.  The output is placed in the
troll.m file, and to check the results, run the troll.m script in MATLAB.
